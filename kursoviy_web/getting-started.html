Поширені запитання

Скільки часу потрібно для налаштування?

Базове налаштування займає 15-30 хвилин. Повна інтеграція з усіма джерелами даних - 1-2 дні. Наші фахівці допоможуть вам на кожному етапі.

Чи потрібні технічні знання для роботи з платформою?

Ні, наш інтуїтивний інтерфейс зрозумілий навіть новачкам. Ми також надаємо детальну документацію та відео-уроки для кожного функціоналу.

Які джерела даних підтримуються платформою?

Ми підтримуємо Google Analytics, Meta Business, CRM системи (Salesforce, HubSpot), бази даних, API та багато інших джерел. Повний список доступний в документації.

Чи є безкоштовний trial та який його термін?

Так, ми пропонуємо 14-денний безкоштовний trial з повним функціоналом. Після закінчення trial ви можете обрати підходящий тарифний план.

Як відбувається підтримка клієнтів?

Ми надаємо цілодобову підтримку через чат, email та телефон. Також проводимо регулярні вебінари та оновлюємо базу знань.

Чи можна інтегрувати платформу з нашими існуючими системами?

Так, наша платформа має гнучкі API для інтеграції з будь-якими корпоративними системами. Наші технічні фахівці допоможуть з налаштуванням.
